fun search(query: String, allowReferences: Boolean = false): List<SearchResult>
Content copied to clipboard

querys Wikipedia, returning reference pages only if you allowReferences.

This function can also be used to suggest pages. For example, if query is "appl", pages about the fruit (apple), the tech company (Apple Inc.), etc. will be returned.

suspend fun search(limit: Int = 2, allowReferences: Boolean = false): List<SearchResult>
Content copied to clipboard

Search for no more than limit (at most 500) random pages and returns reference pages if you allowReferences.

Results are limited to 2 by default in case you allowReferences and the first result is a reference page.
