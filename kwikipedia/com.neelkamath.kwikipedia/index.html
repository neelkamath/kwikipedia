Package com.neelkamath.kwikipedia

Certain Wikipedia pages (e.g., Go) are solely meant to list pages of the same name. This library refers to these pages as "reference pages".

Types

Page
Link copied to clipboard
typealias Page = Map<String, String>
Content copied to clipboard

A Wikipedia page's contents.

Headings (e.g., "Bibliography") are mapped to their respective contents. Since the page's introduction doesn't have a heading, the page's title (e.g., "Apple Inc.") is used instead.
